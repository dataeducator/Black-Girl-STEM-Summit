# Black Girl STEM Summit — Viral Trend Voting App

An interactive Streamlit web app built for the **Black Girl STEM Summit** at **Meharry Medical College** in Nashville, TN. Summit attendees vote on which viral trend they think went the most viral, then reveal the real rankings with data visualizations.

## Features

- **Vote on viral trends** — Amapiano, Marching Band, Step Team, Trombone Solo, and Wash Day
- **QR codes & direct links** — Scan or tap to watch each trend on TikTok/YouTube
- **Live vote tally** — See how the group voted in real time
- **Reveal the real rankings** — Compare the group's predictions against actual view counts
- **Interactive charts** — Slope chart, grouped bar chart, lollipop chart, and per-trend accuracy cards powered by Plotly
- **Meharry Medical College branding** — Maroon & gold palette with Nunito Sans typography

## Getting Started

### Prerequisites

- Python 3.9+

### Installation

```bash
pip install -r requirements.txt
```

### Run the App

```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`.

## Tech Stack

- [Streamlit](https://streamlit.io/) — Web framework
- [Plotly](https://plotly.com/python/) — Interactive visualizations
